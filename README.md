# ติดตั้งโมดูลเพิ่มเติม
### `npm install express`



## วิธีใช้งาน Frontend
ไปที่หน้าโฟลเดอร์หลัก
### `npm run start`
แสดงผลที่ [http://localhost:3000](http://localhost:3000)

## วิธีใช้งาน Backend
ไปที่โฟล์เดอร์ Websocket
### `node server.js`
มีการใช้ API ที่ Port 3001
มีการใช้ Websocket ที่ Port 8080
